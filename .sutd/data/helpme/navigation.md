# Navigation

## Quick cd
- `..` — go up one dir
- `...` — go up two
- `....` — go up three

## Bookmarks
- `bookmark <name>` — save current dir
- `go <name>` — jump to bookmark
- `go` — list all bookmarks
- `unbookmark <name>` — remove

## Examples
    cd /var/www/api
    bookmark api
    cd /
    go api          # back to /var/www/api
